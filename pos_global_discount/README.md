POS Global Discount (Odoo 19)
==============================
Description
===========
POS Global Discount allows cashiers to apply a global discount on the entire Point of Sale order in Odoo 19.
The discount can be applied as a percentage or as a fixed amount using a configured discount product, with optional maximum limits.

Features
=========
Global discount on POS orders
Percentage or fixed amount discount
Uses a discount product
Maximum discount percentage limit
Maximum discount amount limit
POS validation before opening session
Compatible with Odoo 19 POS

Configuration
=============
Go to Point of Sale → Configuration → Point of Sale
Open a POS configuration
Enable Enable Global Discount
Select a Global Discount Product
Set Max Discount % and Max Discount Amount (optional)
If global discount is enabled and no discount product is set, the POS session will not open.

Usage
==========
Open POS session
Add products to the order
Click Global Discount
Choose discount type (Percentage or Amount)
Enter discount value
Discount is applied to the entire order

Notes
==========
Recommended discount product type: Service
Discount product price should be 0.00
Max Discount Amount = 0 means no limit
Compatibility
===============
Odoo 19.0
